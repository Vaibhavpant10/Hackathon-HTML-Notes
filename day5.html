-Header Section

In HTML, the header section refers to the part of the webpage
that contains introductory or navigational content.
This is usually created using the header tag.

-Main Section

In HTML, the main element (often referred to as the "Main Section") is a semantic HTML5 element used to define the primary content of a webpage.
It represents the dominant content of the body of a document, excluding headers, footers, navigation bars, and sidebars.
The main element helps improve accessibility and SEO by clearly indicating the core content to browsers, screen readers, and search engines.

-Adding Controls to Audio

In HTML, "Adding Controls to Audio" refers to using the controls attribute on the audio element to display a built-in user interface for controlling audio playback.
This attribute enables default browser-provided controls, such as play/pause buttons, seek bar, volume slider, and sometimes fullscreen or download options.
Without the controls attribute, the audio plays automatically (if autoplay is set) but remains invisible and uncontrollable to the user.
