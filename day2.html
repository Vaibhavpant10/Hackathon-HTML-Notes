- Introduction to HTML Tables

HTML tables are used to organize and display data in a tabular format on web pages.
They consist of rows and columns, allowing you to present information in a structured manner.
Tables are created using the `table` element, with rows defined
by the `` (table row) element and cells defined by the `td` (table data) element for regular cells and `th` (table header) element for header cells.

- Rows and Columns

In HTML, "rows" and "columns" are fundamental concepts
primarily used in the context of tables to organize and display data in a structured format.

  • Rows (tr): A row represents a horizontal grouping of data cells within a table.
    In HTML, table rows are defined using the tr (table row) element.
    Each tr element contains one or more th (table header) or td (table data) elements, which represent the individual cells within that row.
  • Columns: Columns represent a vertical grouping of data cells within a table.
    While there isn't a specific HTML element like column to define a column directly, columns are implicitly formed by the arrangement of th or td elements across the rows.
    The first cell in each row forms the first column, the second cell in each row forms the second column, and so on.

Header cells are typically used to label the columns or rows of a table, providing context and meaning to the data contained within the table.

- Understanding Colspan

The colspan attribute in HTML is used within td (table data) or th (table header) tags to specify
how many columns a single cell should span horizontally within an HTML table.
By default, each cell occupies only one column.

- Understanding Rowspan

The rowspan attribute in HTML is used within table cells (td or th) to specify how many rows a particular cell should span or merge vertically.
It allows a single cell to occupy the space of multiple rows, effectively combining them into a larger cell.

- Table Header, Body, and Footer

Table Header:- In HTML, a table header refers to the element that defines a header cell within a table. This is primarily done using the th (table header) tag.

Body:- The body tag in HTML defines the main content of an HTML document.
It is a fundamental element that contains all the visible
content of a web page, such as text, images, links, tables, lists, and other elements that are displayed in a web browser.

Footer:- The HTML footer tag defines the footer section of a document
or a specific section within a document.
It is a semantic HTML5 element,
meaning it conveys meaning about the content it encloses, improving document structure and accessibility.
