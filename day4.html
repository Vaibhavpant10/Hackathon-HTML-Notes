-what is an HTML Form.

An HTML form is a section of a web page that contains interactive elements, such as text fields, checkboxes, radio buttons, and submit buttons.
Forms are used to collect user input and send it to a server for processing.
They are commonly used for tasks such as user registration, login, feedback submission, and online purchases.

-Form Tag

The form tag is used to create an HTML form. It acts as a container for all the form elements and defines how the form data will be submitted.
The form tag has several attributes, including action, method, and enctype.

-Name & Value Attributes

The name attribute is used to specify a unique name for each form element. This name is used to identify the form data when it is submitted to the server.
The value attribute is used to specify the default value of a form element.

-Radio Buttons

Radio buttons are used to allow users to select one option from a group of options.
They are created using the input tag with the type attribute set to "radio".
All radio buttons in a group should have the same name attribute to ensure that only one option can be selected at a time.

Checkboxes

Checkboxes are graphical user interface (GUI) elements used in forms, surveys, or applications to allow users to select one or more options from a list.
They're typically represented as small square boxes that can either be checked (selected) or unchecked (deselected).

-Textarea Tag

The textarea tag in HTML is used to create a multi-line text input field within a form.
It allows users to enter and edit larger amounts of text compared to a standard single-line input field.

-Select Tag

The select tag in HTML is used to create a drop-down list or a list box within a form.
It allows users to choose one or more options from a predefined set of choices.

-option Tag

The option tag in HTML is used to define individual options within a select element (drop-down list or list box).
Each option represents a choice that users can select from the list.

-Button Tag

The button tag in HTML is used to create clickable buttons on a web page.
Buttons can be used to submit forms, trigger JavaScript functions, or perform various actions when clicked by the user.
